Portfolio Optimisation

.

This tool is based on Efficient Frontier of Modern Portfolio theory. The efficient frontier is the set of optimal portfolios that offer the highest expected return for a defined level of risk or the lowest risk for a given level of expected return. Portfolios that lie below the efficient frontier are sub-optimal because they do not provide enough return for the level of risk. The plot shows all the portfolio constructed. Tables show the weightages of the stocks to be bought for the desired portfolio type (Here choosing between Max Sharpe or Low Volatility). The weightage together sum to be 1(or 100%), so you can make decisions about the amount of companies to be bought accordingly.
